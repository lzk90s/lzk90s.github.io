# 电商运营

电商运营是通过运营手段，提升电商平台的用户价值、商品价值和商业价值的运营工作。

## 电商运营核心

### 电商的本质

- **人**：用户运营
- **货**：商品运营
- **场**：场景运营

### 电商运营目标

1. **GMV增长**：提升成交总额
2. **用户增长**：获取更多用户
3. **转化提升**：提高转化率
4. **复购提升**：提高用户复购率
5. **客单价提升**：提高平均订单金额

## 用户运营

### 用户分层

1. **新用户**
   - 目标：完成首单
   - 策略：新人专享、首单优惠

2. **活跃用户**
   - 目标：提升复购
   - 策略：个性化推荐、会员权益

3. **沉睡用户**
   - 目标：唤醒用户
   - 策略：召回活动、优惠刺激

4. **流失用户**
   - 目标：挽回用户
   - 策略：流失分析、针对性挽回

### 用户运营策略

1. **会员体系**
   - 会员等级
   - 会员权益
   - 积分体系

2. **个性化推荐**
   - 基于用户画像
   - 基于行为数据
   - 提升转化率

3. **用户触达**
   - 推送通知
   - 短信营销
   - 邮件营销
