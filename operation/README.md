# 运营营销领域（1-100）

运营是连接产品和用户的桥梁，关注从1到100的过程，通过一系列手段提升用户价值、产品价值和商业价值。

## 运营领域的核心

**运营 = 从1到100的过程**

- **1**：产品上线、MVP验证完成
- **100**：用户规模增长、商业价值实现、产品成熟

运营的核心是在产品已经验证的基础上，通过拉新、促活、留存、转化、传播等手段，实现用户和业务的快速增长。

## 运营的核心目标

1. **拉新**：获取新用户
2. **促活**：提升用户活跃度
3. **留存**：提高用户 retention
4. **转化**：实现商业价值转化
5. **传播**：促进用户自传播

## 运营的底层逻辑

- **用户价值**：为用户创造价值
- **产品价值**：提升产品价值
- **商业价值**：实现商业目标
