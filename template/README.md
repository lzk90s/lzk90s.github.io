# 模板

每种文档有相对应的面向的角色，根据对象的不同，文档会有不同

## BRD: business requirement document

面向的角色

- CEO
- COO
- CFO
- CTO

一般可以采用 PPT 方式

需要包含的内容

## MRD: market requirement document

面向的角色

- 销售 BD

## PRD: product requirement document

面向的角色

- UI，UE
- 前端开发
- 后端开发
- 测试

## TDD: technology design document

- 前端开发
- 后端开发
- 测试
